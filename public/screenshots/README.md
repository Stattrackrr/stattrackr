## Image Requirements
- **Mobile**: Recommended size ~300x650px (iPhone 17 aspect ratio)
- **Desktop**: Recommended size ~900x600px (16:9 aspect ratio)
- Format: PNG or JPG
- Optimize images for web (keep file sizes reasonable)

The homepage will automatically use these images when they're available, and fall back to placeholder icons if they're missing.
